About the Brand Concierge: Hackathon Edition

The Brand Concierge AI was developed to automate daily work by providing a single, intelligent source of truth for design guidelines. This project helps eliminate the time spent searching for assets and rules, allowing stakeholders to easily find information and simplify complex problems related to brand compliance.

Technical Execution:

  • Knowledge Base: Team guidelines are sourced from a dynamic, AI-curated knowledge repository.
  • Gemini 2.5 Flash API: Used for hyper-efficient structured JSON output (reply, status, and link), ensuring reliable data parsing on the frontend.
  • Empathy Prompting: The system instruction guides the AI to reply with a warm, supportive, and conversational tone, turning a static document into a friendly assistant.
  • Proxy Architecture: A Vercel Node.js Proxy securely handles the API key and file reading, ensuring stable deployment even with complex environment constraints.

Future Roadmap: Aligned with Hackathon Criteria

The following features represent the next stages of development, showing how the Brand Concierge provides clear value against the hackathon judging criteria.

Innovation & Creativity

Goal: Pushing the idea further by moving beyond text retrieval.

  • AI Visual Compliance Check: Allow users to upload a design and ask, "Does this color match brand guidelines?" The AI would use Image Understanding to verify colors and fonts in the image against the internal knowledge base rules.

Execution & Usability

Goal: Automate daily work and ensure the solution is robust and usable.

  • Live Source Integration: Replace the static knowledge file with a direct, automated fetch from Figma, Notion, or Confluence via their APIs, ensuring the knowledge is always up-to-date without manual intervention.

Impact & ROI

Goal: Quantifiable time savings and risk reduction for the marketing team.

  • Multi-User Authentication & Logging: Implement secure login and track usage metrics (e.g., "AI saved 50 hours of designer search time this month"). This allows for a clear ROI calculation to justify scaling the project.
